Migrasi Data
Import data dari sistem Netlify lama ke Supabase
🔄 Opsi A: Import Otomatis dari Netlify
Hubungkan langsung ke site Netlify lama dan import semua data secara otomatis. Pastikan site Netlify lama sudah di-update dengan patch migrasi.
1
Deploy patch migrasi ke Netlify lama (lihat file netlify-migration-patch.zip)
2
Masukkan URL Netlify lama dan password admin
3
Klik Import — sistem akan mengambil semua data secara otomatis
URL Site Netlify Lama
Password Admin Netlify Lama
📁 Opsi B: Import dari JSON File
Export data dari Netlify lama sebagai JSON, lalu paste di sini untuk import ke Supabase.
1
Login ke Netlify backoffice lama → Migrasi → Export Data JSON
2
Copy seluruh isi JSON yang didownload
3
Paste di bawah ini dan klik Import
JSON Export Data
ℹ️ Tentang Migrasi
• Data yang akan dimigrasikan: Akun pengguna, riwayat makan (beserta foto & data gizi), dan laporan kendala
• Duplikat data otomatis diabaikan (aman untuk dijalankan berkali-kali)
• Proses migrasi bisa memakan waktu 1–5 menit tergantung jumlah data
• Backup data Supabase sebelum migrasi jika ada data penting
